CSS

Def: CSS - Cascading Styling Sheet: 계단식으로 외모를 부여하는 표
(계단식 즉 CSS 적용 우선순위: 중요도(!important)) > 명시도(.parent .child{}) > 순서도(.parent1.parent2) Simple: 웹의 스킨/외모

Structure: Selector(선택자)
+ 선택자 구분: selector(선택자) Declaration(선언 오픈){ property(속성) : value(값) }(선언 클로즈)
- 선택자:
- 자식 선택자 (.list > p{}) ---> (.list p{})
- 제외 선택자 (.list p:not(.eight), .list p:not(:nth-child(5)))
- 순번 선택자 (.list:first-child, .list:last-child, list:nth-child(5), list:nth-child(odd), list:nth-child(event))
- 가상 선택자 (.list::after { content: ''; display: block clear: both; } //불필요하게 tag 생성 없이 기존 tag로float제거
- 가상 선택자2 (input[type="radio"]:checked ~.content{display: block;} or input[type="radio"]:checked + label + .content{display: block;} //CSS Tab 만들기

- 속성:
- 순서 정리 속성
- float (넓이 배치) *상위 요소에 float을 정해줘야한다*
- clear: bloth (float 해제) //가상 선택자 사용
- overflow: //콘텐츠를 어떻게 감쌀것인지 지정
- position: static(기본값), relative, fixed(고정값), relative > absolute
- display: none:block:inline-block; (감추기 || 보이기 || horizontal line으로 바꾸기)
- vertical & horizontal nav > ul {list-style: none} li {float: left || list-style: none}

- 화면 크기별 반응 속성
